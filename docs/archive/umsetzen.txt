Alles klar — hier ist ein kompaktes Spec/ADR, das du direkt als Grundlage nehmen kannst. (High-level Commands, binär, backend-fähig, Worker-ready.)

⸻

ADR: Loom Render Command Stream v1

Ziel
	•	Rust bleibt Source of Truth (State, Interaction, Tools).
	•	Rendering läuft über High-level RenderCommands in einem binären Command-Buffer.
	•	JS ist Backend-Treiber (Canvas2D heute, später WebGL/WebGPU/OffscreenCanvas).
	•	Pro Animation-Frame: 1× InputBatch rein, 1× CommandBuffer raus.

Nicht-Ziele (v1)
	•	Kein Vertex-/GPU-Drawcall-Stream.
	•	Kein vollständiger Text-Layout-Engine in Rust.
	•	Kein “alles im Worker” sofort (aber kompatibel).

⸻

Frame-Pipeline

JS → WASM
	•	push_events(ptr,len) (oder push_events_u8(typedArray)): EventBatch für diesen Frame
	•	tick(dt_ms: f32) optional
	•	render(width, height, pixel_ratio)

WASM → JS
	•	get_cmd_ptr() -> *const u8
	•	get_cmd_len() -> usize
	•	get_frame_meta() -> FrameMeta (tiny struct, siehe unten)

JS parst cmd_buffer und führt Commands aus.

⸻

Command Buffer Format

Header

bytes 0..3   magic: u32  = 0x4C4F4F4D   // "LOOM"
bytes 4..5   version: u16 = 1
bytes 6..7   flags: u16   // reserved
bytes 8..11  cmd_count: u32 (optional, 0 wenn unknown)
bytes 12..15 text_count: u32
bytes 16..19 text_table_offset: u32
bytes 20..23 cmd_stream_offset: u32

	•	Alles little-endian
	•	cmd_stream_offset zeigt auf den Command-Stream
	•	text_table_offset zeigt auf Text-Table (siehe unten)

Command Record

Jeder Command:

opcode: u8
size:   u16   // payload bytes (ohne opcode+size)
payload: [size bytes]

Parsing in JS: lese opcode, size, payload, spring weiter.

⸻

Opcodes (v1)

Frame / Layer
	•	0x01 BeginFrame {} (optional, wenn header reicht)
	•	0x02 Clear { rgba: u32 }
	•	0x03 BeginLayer { layer_id: u16 }
	•	0x04 EndLayer {}
	•	0x05 SetClipRect { x:f32, y:f32, w:f32, h:f32 }
	•	0x06 ResetClip {}

Layer-IDs z.B.:
	•	0 = background/grid
	•	1 = candles
	•	2 = indicators
	•	3 = drawings
	•	4 = overlay (crosshair/handles)
	•	5 = debug

Styles (stateful)
	•	0x10 SetStroke { rgba:u32, width:f32, dash_id:u16 }
	•	0x11 SetFill { rgba:u32 }
	•	0x12 SetFont { font_id:u16, size:f32, weight:u16 }
	•	0x13 SetTextAlign { align:u8, baseline:u8 }
	•	0x14 SetAlpha { alpha:f32 } (optional)

Konzept: JS Backend hält aktuellen “Paint State”. So sparst du Payload.

Primitives
	•	0x20 DrawLine { x1:f32, y1:f32, x2:f32, y2:f32 }
	•	0x21 DrawRect { x:f32, y:f32, w:f32, h:f32 }
	•	0x22 FillRect { x:f32, y:f32, w:f32, h:f32 }
	•	0x23 DrawCircle { x:f32, y:f32, r:f32 }
	•	0x24 FillCircle { x:f32, y:f32, r:f32 }
	•	0x25 DrawPolyline { points_offset:u32, points_len:u32 }
	•	0x26 FillPolygon { points_offset:u32, points_len:u32 } (optional)
	•	0x27 DrawText { x:f32, y:f32, text_id:u32 }

Polyline Points Buffer
	•	liegt im selben cmd_buffer hinter den Commands oder in separatem “arena buffer”.
	•	Format: f32 x, f32 y repeated (2 * points_len floats)
	•	points_offset ist Byte-Offset ab Buffer-Start.

Debug (optional, extrem hilfreich)
	•	0xF0 DebugPoint { x:f32,y:f32, rgba:u32, r:f32 }
	•	0xF1 DebugRect { x:f32,y:f32,w:f32,h:f32, rgba:u32 }

⸻

Text Table (pro Frame)

Ziel: keine Strings in jedem DrawText.

Text Table Record:

text_id: u32
font_id: u16
flags:   u16
utf8_len: u32
utf8_bytes: [utf8_len]
padding to 4 bytes

JS Backend baut intern eine Map text_id -> string für dieses Frame.

Font IDs
	•	0 = default UI font
	•	1 = monospace
	•	später erweiterbar

⸻

FrameMeta (separat, klein, nicht im Command-Stream)

Rückgabe als WASM-Export (struct):
	•	dirty_flags: u32 (bitset: FULL=1, OVERLAY=2, CURSOR=4, NONE=0)
	•	cursor_hint: u8 (0 default, 1 crosshair, 2 ns-resize, 3 ew-resize, 4 move, …)
	•	hover_id: u32 (0 = none)
	•	hover_handle: u16 (0 = none)
	•	selection_count: u16
	•	optional: selection_ptr/len (oder per extra getter)

Warum separat?
	•	UI muss diese Infos nutzen, ohne den RenderStream zu parsen.

⸻

Input Event Batch (JS → WASM)

Auch binär, pro Frame:

Header:
	•	count: u16
	•	reserved: u16

Event Record:
	•	etype: u8 (pointer/key/wheel)
	•	size: u16
	•	payload

Minimal v1 Payloads:
	•	PointerMove/Down/Up:
	•	x:f32 y:f32 buttons:u16 mods:u16 pointer_id:u16
	•	Wheel:
	•	dx:f32 dy:f32 mods:u16
	•	KeyDown/Up:
	•	key_code:u16 mods:u16 repeat:u8

Mods bitset: Shift=1, Alt=2, Ctrl=4, Meta=8

⸻

Invalidation-Regeln (wichtig)

Rust entscheidet “was muss neu”:
	•	Cursor-only: nur Overlay Layer rendern
	•	Overlay: drawings handles/crosshair
	•	Full: candles/indicators/etc.

JS kann trotzdem rAF laufen lassen, aber wenn dirty_flags == NONE, kann es skippen.

⸻

Kompatibilität Worker/OffscreenCanvas

Wenn du Commands + Events binär hast, ist der spätere Sprung leicht:
	•	JS Main Thread sammelt UI events → postMessage(ArrayBuffer) an Worker
	•	Worker läuft WASM + Backend (OffscreenCanvas)
	•	Worker rendert direkt (keine DOM)
	•	Für Text: entweder Worker-Canvas Text oder weiterhin DOM overlay (Main Thread)

⸻

“Done”-Kriterien für v1
	•	1 Panel, Candles + Crosshair werden über RenderCommands gezeichnet
	•	Mousemove erzeugt nur Overlay invalidation
	•	Trendline Tool:
	•	add drawing (2 points)
	•	hit test handles
	•	drag handle
	•	undo/redo (AddTrendline, MoveHandle)

render_protocol.rs

// crates/chartcore/src/render_protocol.rs
#![allow(dead_code)]

#[repr(u8)]
#[derive(Copy, Clone)]
pub enum Op {
    BeginFrame   = 0x01,
    Clear        = 0x02,
    BeginLayer   = 0x03,
    EndLayer     = 0x04,
    SetClipRect  = 0x05,
    ResetClip    = 0x06,

    SetStroke    = 0x10,
    SetFill      = 0x11,
    SetFont      = 0x12,
    SetTextAlign = 0x13,
    SetAlpha     = 0x14,

    DrawLine     = 0x20,
    DrawRect     = 0x21,
    FillRect     = 0x22,
    DrawCircle   = 0x23,
    FillCircle   = 0x24,
    DrawPolyline = 0x25,
    FillPolygon  = 0x26,
    DrawText     = 0x27,

    DebugPoint   = 0xF0,
    DebugRect    = 0xF1,
}

#[repr(C)]
#[derive(Copy, Clone, Default)]
pub struct FrameMeta {
    pub dirty_flags: u32,   // FULL=1, OVERLAY=2, CURSOR=4
    pub cursor_hint: u8,    // 0 default, 1 crosshair, 2 move, etc.
    pub _pad0: [u8; 3],
    pub hover_id: u32,      // 0 none
    pub hover_handle: u16,  // 0 none
    pub selection_count: u16,
}

pub const MAGIC: u32 = 0x4D4F4F4C; // "LOOM" little-endian

#[repr(C)]
#[derive(Copy, Clone, Default)]
pub struct CmdHeader {
    pub magic: u32,
    pub version: u16,
    pub flags: u16,
    pub cmd_count: u32,
    pub text_count: u32,
    pub text_table_offset: u32,
    pub cmd_stream_offset: u32,
}

pub struct CmdWriter {
    buf: Vec<u8>,
    // Offsets recorded after building
    header_pos: usize,
    cmd_count: u32,
    text_count: u32,
    text_table_offset: u32,
    cmd_stream_offset: u32,
}

impl CmdWriter {
    pub fn new(capacity: usize) -> Self {
        let mut buf = Vec::with_capacity(capacity);

        // Reserve header space (filled later)
        let header_pos = 0;
        buf.resize(std::mem::size_of::<CmdHeader>(), 0);

        Self {
            buf,
            header_pos,
            cmd_count: 0,
            text_count: 0,
            text_table_offset: 0,
            cmd_stream_offset: 0,
        }
    }

    #[inline] pub fn as_ptr(&self) -> *const u8 { self.buf.as_ptr() }
    #[inline] pub fn len(&self) -> usize { self.buf.len() }
    #[inline] pub fn clear(&mut self) {
        self.buf.truncate(std::mem::size_of::<CmdHeader>());
        self.cmd_count = 0;
        self.text_count = 0;
        self.text_table_offset = 0;
        self.cmd_stream_offset = 0;
    }

    fn write_u8(&mut self, v: u8) { self.buf.push(v); }
    fn write_u16(&mut self, v: u16) { self.buf.extend_from_slice(&v.to_le_bytes()); }
    fn write_u32(&mut self, v: u32) { self.buf.extend_from_slice(&v.to_le_bytes()); }
    fn write_f32(&mut self, v: f32) { self.buf.extend_from_slice(&v.to_le_bytes()); }

    fn align4(&mut self) {
        while self.buf.len() % 4 != 0 { self.buf.push(0); }
    }

    /// Start command stream (after text table is written). Call once.
    pub fn begin_cmd_stream(&mut self) {
        self.align4();
        self.cmd_stream_offset = self.buf.len() as u32;
    }

    pub fn begin_text_table(&mut self) {
        self.align4();
        self.text_table_offset = self.buf.len() as u32;
    }

    /// Text record: [text_id:u32][font_id:u16][flags:u16][len:u32][utf8...][pad]
    pub fn define_text(&mut self, text_id: u32, font_id: u16, flags: u16, utf8: &str) {
        self.write_u32(text_id);
        self.write_u16(font_id);
        self.write_u16(flags);
        let bytes = utf8.as_bytes();
        self.write_u32(bytes.len() as u32);
        self.buf.extend_from_slice(bytes);
        self.align4();
        self.text_count += 1;
    }

    fn cmd_begin(&mut self, op: Op, payload_size: u16) -> usize {
        self.write_u8(op as u8);
        self.write_u16(payload_size);
        self.cmd_count += 1;
        self.buf.len()
    }

    pub fn clear_rgba(&mut self, rgba: u32) {
        self.cmd_begin(Op::Clear, 4);
        self.write_u32(rgba);
    }

    pub fn begin_layer(&mut self, layer_id: u16) {
        self.cmd_begin(Op::BeginLayer, 2);
        self.write_u16(layer_id);
    }

    pub fn end_layer(&mut self) {
        self.cmd_begin(Op::EndLayer, 0);
    }

    pub fn set_clip_rect(&mut self, x: f32, y: f32, w: f32, h: f32) {
        self.cmd_begin(Op::SetClipRect, 16);
        self.write_f32(x); self.write_f32(y); self.write_f32(w); self.write_f32(h);
    }

    pub fn reset_clip(&mut self) {
        self.cmd_begin(Op::ResetClip, 0);
    }

    /// Stroke: rgba(u32), width(f32), dash_id(u16)
    pub fn set_stroke(&mut self, rgba: u32, width: f32, dash_id: u16) {
        self.cmd_begin(Op::SetStroke, 4 + 4 + 2);
        self.write_u32(rgba);
        self.write_f32(width);
        self.write_u16(dash_id);
    }

    pub fn set_fill(&mut self, rgba: u32) {
        self.cmd_begin(Op::SetFill, 4);
        self.write_u32(rgba);
    }

    /// Font: font_id(u16), size(f32), weight(u16)
    pub fn set_font(&mut self, font_id: u16, size: f32, weight: u16) {
        self.cmd_begin(Op::SetFont, 2 + 4 + 2);
        self.write_u16(font_id);
        self.write_f32(size);
        self.write_u16(weight);
    }

    /// Text align: align(u8), baseline(u8)
    pub fn set_text_align(&mut self, align: u8, baseline: u8) {
        self.cmd_begin(Op::SetTextAlign, 2);
        self.write_u8(align);
        self.write_u8(baseline);
    }

    pub fn draw_line(&mut self, x1: f32, y1: f32, x2: f32, y2: f32) {
        self.cmd_begin(Op::DrawLine, 16);
        self.write_f32(x1); self.write_f32(y1); self.write_f32(x2); self.write_f32(y2);
    }

    pub fn draw_rect(&mut self, x: f32, y: f32, w: f32, h: f32) {
        self.cmd_begin(Op::DrawRect, 16);
        self.write_f32(x); self.write_f32(y); self.write_f32(w); self.write_f32(h);
    }

    pub fn fill_rect(&mut self, x: f32, y: f32, w: f32, h: f32) {
        self.cmd_begin(Op::FillRect, 16);
        self.write_f32(x); self.write_f32(y); self.write_f32(w); self.write_f32(h);
    }

    pub fn draw_text(&mut self, x: f32, y: f32, text_id: u32) {
        self.cmd_begin(Op::DrawText, 12);
        self.write_f32(x); self.write_f32(y); self.write_u32(text_id);
    }

    pub fn finish(&mut self) {
        // Write header
        let header = CmdHeader {
            magic: MAGIC,
            version: 1,
            flags: 0,
            cmd_count: self.cmd_count,
            text_count: self.text_count,
            text_table_offset: self.text_table_offset,
            cmd_stream_offset: self.cmd_stream_offset,
        };

        let bytes = unsafe {
            std::slice::from_raw_parts(
                (&header as *const CmdHeader) as *const u8,
                std::mem::size_of::<CmdHeader>(),
            )
        };

        self.buf[self.header_pos..self.header_pos + bytes.len()].copy_from_slice(bytes);
    }
}

WASM Exports (skizziert)

// crates/chartcore/src/wasm_api.rs
use wasm_bindgen::prelude::*;
use crate::render_protocol::{CmdWriter, FrameMeta};

thread_local! {
    static WRITER: std::cell::RefCell<CmdWriter> =
        std::cell::RefCell::new(CmdWriter::new(256 * 1024));
    static META: std::cell::Cell<FrameMeta> = std::cell::Cell::new(FrameMeta {
        dirty_flags: 1, cursor_hint: 0, _pad0: [0;3], hover_id: 0, hover_handle: 0, selection_count: 0
    });
}

#[wasm_bindgen]
pub fn render_frame(width: u32, height: u32, pixel_ratio: f32) {
    let _ = (width, height, pixel_ratio);

    WRITER.with(|w| {
        let mut w = w.borrow_mut();
        w.clear();

        // 1) text table
        w.begin_text_table();
        w.define_text(1, 0, 0, "LOOM");
        w.define_text(2, 0, 0, "Trendline");

        // 2) commands
        w.begin_cmd_stream();
        w.clear_rgba(0x101418FF); // RGBA packed, interpret in JS (see below)

        w.begin_layer(3); // drawings
        w.set_stroke(0xFFCC33FF, 1.0, 0); // stroke
        w.draw_line(50.0, 50.0, 250.0, 120.0);
        w.set_fill(0xFFCC33FF);
        w.fill_rect(48.0, 48.0, 4.0, 4.0); // handle
        w.fill_rect(248.0, 118.0, 4.0, 4.0); // handle
        w.end_layer();

        w.begin_layer(4); // overlay
        w.set_font(0, 12.0, 400);
        w.set_text_align(0, 0);
        w.draw_text(12.0, 18.0, 1);
        w.end_layer();

        w.finish();
    });

    META.with(|m| {
        let mut meta = m.get();
        meta.dirty_flags = 1; // FULL
        m.set(meta);
    });
}

#[wasm_bindgen]
pub fn get_cmd_ptr() -> *const u8 {
    WRITER.with(|w| w.borrow().as_ptr())
}

#[wasm_bindgen]
pub fn get_cmd_len() -> usize {
    WRITER.with(|w| w.borrow().len())
}

#[wasm_bindgen]
pub fn get_frame_meta() -> FrameMeta {
    META.with(|m| m.get())
}

2) TypeScript: Parser + Canvas2D Backend

canvas_backend.ts

type TextTable = Map<number, string>;

const MAGIC = 0x4D4F4F4C; // "LOOM" little-endian

enum Op {
  BeginFrame   = 0x01,
  Clear        = 0x02,
  BeginLayer   = 0x03,
  EndLayer     = 0x04,
  SetClipRect  = 0x05,
  ResetClip    = 0x06,

  SetStroke    = 0x10,
  SetFill      = 0x11,
  SetFont      = 0x12,
  SetTextAlign = 0x13,
  SetAlpha     = 0x14,

  DrawLine     = 0x20,
  DrawRect     = 0x21,
  FillRect     = 0x22,
  DrawCircle   = 0x23,
  FillCircle   = 0x24,
  DrawPolyline = 0x25,
  FillPolygon  = 0x26,
  DrawText     = 0x27,

  DebugPoint   = 0xF0,
  DebugRect    = 0xF1,
}

// Interpret u32 RGBA into CSS rgba()
function rgbaU32ToCss(rgba: number): string {
  const r = (rgba >>> 24) & 0xff;
  const g = (rgba >>> 16) & 0xff;
  const b = (rgba >>> 8) & 0xff;
  const a = (rgba >>> 0) & 0xff;
  return `rgba(${r},${g},${b},${(a / 255)})`;
}

function align4(n: number): number {
  return (n + 3) & ~3;
}

export class LoomCanvas2DBackend {
  private ctx: CanvasRenderingContext2D;
  private texts: TextTable = new Map();

  constructor(ctx: CanvasRenderingContext2D) {
    this.ctx = ctx;
  }

  execute(buffer: Uint8Array) {
    const dv = new DataView(buffer.buffer, buffer.byteOffset, buffer.byteLength);

    // ---- Header ----
    const magic = dv.getUint32(0, true);
    if (magic !== MAGIC) return;

    const version = dv.getUint16(4, true);
    if (version !== 1) return;

    const cmdCount = dv.getUint32(8, true);
    const textCount = dv.getUint32(12, true);
    const textTableOffset = dv.getUint32(16, true);
    const cmdStreamOffset = dv.getUint32(20, true);

    // ---- Text table ----
    this.texts.clear();
    let p = textTableOffset;
    for (let i = 0; i < textCount; i++) {
      const textId = dv.getUint32(p + 0, true);
      const fontId = dv.getUint16(p + 4, true);
      const flags  = dv.getUint16(p + 6, true);
      const len    = dv.getUint32(p + 8, true);
      const bytes = buffer.subarray(p + 12, p + 12 + len);
      const str = new TextDecoder().decode(bytes);
      this.texts.set(textId, str);

      // currently unused:
      void fontId; void flags;

      p = align4(p + 12 + len);
    }

    // ---- Command stream ----
    let c = cmdStreamOffset;
    for (let i = 0; i < cmdCount; i++) {
      const opcode = dv.getUint8(c);
      const size = dv.getUint16(c + 1, true);
      const payloadStart = c + 3;

      switch (opcode) {
        case Op.Clear: {
          const rgba = dv.getUint32(payloadStart, true);
          this.ctx.save();
          this.ctx.setTransform(1,0,0,1,0,0);
          this.ctx.globalAlpha = 1;
          this.ctx.fillStyle = rgbaU32ToCss(rgba);
          this.ctx.fillRect(0, 0, this.ctx.canvas.width, this.ctx.canvas.height);
          this.ctx.restore();
          break;
        }

        case Op.BeginLayer: {
          // layer_id: u16
          // const layerId = dv.getUint16(payloadStart, true);
          // You can use this to toggle debug layers, etc.
          break;
        }

        case Op.EndLayer: {
          break;
        }

        case Op.SetClipRect: {
          const x = dv.getFloat32(payloadStart + 0, true);
          const y = dv.getFloat32(payloadStart + 4, true);
          const w = dv.getFloat32(payloadStart + 8, true);
          const h = dv.getFloat32(payloadStart + 12, true);
          this.ctx.save();
          this.ctx.beginPath();
          this.ctx.rect(x, y, w, h);
          this.ctx.clip();
          break;
        }

        case Op.ResetClip: {
          // assumes clips were paired with save()
          this.ctx.restore();
          break;
        }

        case Op.SetStroke: {
          const rgba = dv.getUint32(payloadStart + 0, true);
          const width = dv.getFloat32(payloadStart + 4, true);
          const dashId = dv.getUint16(payloadStart + 8, true);

          this.ctx.strokeStyle = rgbaU32ToCss(rgba);
          this.ctx.lineWidth = width;

          // dash registry (v1: simple)
          if (dashId === 0) this.ctx.setLineDash([]);
          else if (dashId === 1) this.ctx.setLineDash([4, 4]);
          else this.ctx.setLineDash([]);
          break;
        }

        case Op.SetFill: {
          const rgba = dv.getUint32(payloadStart + 0, true);
          this.ctx.fillStyle = rgbaU32ToCss(rgba);
          break;
        }

        case Op.SetFont: {
          const fontId = dv.getUint16(payloadStart + 0, true);
          const sizePx = dv.getFloat32(payloadStart + 2, true);
          const weight = dv.getUint16(payloadStart + 6, true);

          const family = fontId === 1 ? "ui-monospace, SFMono-Regular, Menlo, monospace" : "system-ui, -apple-system, Segoe UI, Roboto, sans-serif";
          this.ctx.font = `${weight} ${sizePx}px ${family}`;
          break;
        }

        case Op.SetTextAlign: {
          const align = dv.getUint8(payloadStart + 0);
          const baseline = dv.getUint8(payloadStart + 1);
          this.ctx.textAlign = align === 0 ? "left" : align === 1 ? "center" : "right";
          this.ctx.textBaseline = baseline === 0 ? "top" : baseline === 1 ? "middle" : "alphabetic";
          break;
        }

        case Op.DrawLine: {
          const x1 = dv.getFloat32(payloadStart + 0, true);
          const y1 = dv.getFloat32(payloadStart + 4, true);
          const x2 = dv.getFloat32(payloadStart + 8, true);
          const y2 = dv.getFloat32(payloadStart + 12, true);
          this.ctx.beginPath();
          this.ctx.moveTo(x1, y1);
          this.ctx.lineTo(x2, y2);
          this.ctx.stroke();
          break;
        }

        case Op.DrawRect: {
          const x = dv.getFloat32(payloadStart + 0, true);
          const y = dv.getFloat32(payloadStart + 4, true);
          const w = dv.getFloat32(payloadStart + 8, true);
          const h = dv.getFloat32(payloadStart + 12, true);
          this.ctx.strokeRect(x, y, w, h);
          break;
        }

        case Op.FillRect: {
          const x = dv.getFloat32(payloadStart + 0, true);
          const y = dv.getFloat32(payloadStart + 4, true);
          const w = dv.getFloat32(payloadStart + 8, true);
          const h = dv.getFloat32(payloadStart + 12, true);
          this.ctx.fillRect(x, y, w, h);
          break;
        }

        case Op.DrawText: {
          const x = dv.getFloat32(payloadStart + 0, true);
          const y = dv.getFloat32(payloadStart + 4, true);
          const textId = dv.getUint32(payloadStart + 8, true);
          const text = this.texts.get(textId) ?? "";
          this.ctx.fillText(text, x, y);
          break;
        }

        default:
          // unknown opcode: ignore
          break;
      }

      c = payloadStart + size;
    }
  }
}

1) EventBatch Spec v1

Header (4 Bytes)
	•	count: u16 Anzahl Events
	•	reserved: u16 (0)

Event Record (variable)
	•	etype: u8
	•	size: u16 payload bytes (ohne etype+size)
	•	payload: [size]

Mods Bitset (u16)
	•	Shift=1, Alt=2, Ctrl=4, Meta=8

Event Types
	•	1 = PointerMove
	•	2 = PointerDown
	•	3 = PointerUp
	•	4 = Wheel
	•	5 = KeyDown
	•	6 = KeyUp

Payloads

Pointer* (Move/Down/Up) payload (16 bytes):
	•	x: f32
	•	y: f32
	•	buttons: u16 (wie PointerEvent.buttons)
	•	mods: u16
	•	pointer_id: u16
	•	kind: u8 (0 mouse, 1 touch, 2 pen)

Wheel payload (12 bytes):

	•	key_code: u16 (z.B. “KeyZ”, “Escape” mapped auf eigene Codes)

// apps/frontend/src/lib/event-batch.ts
export const MOD_SHIFT = 1, MOD_ALT = 2, MOD_CTRL = 4, MOD_META = 8;

export enum EType {
  PointerMove = 1,
  PointerDown = 2,
  PointerUp   = 3,
  Wheel       = 4,
  KeyDown     = 5,
  KeyUp       = 6,
}

type Packed = { buffer: ArrayBuffer; byteLength: number };

function writeU16(dv: DataView, o: number, v: number) { dv.setUint16(o, v, true); }
function writeU8(dv: DataView, o: number, v: number) { dv.setUint8(o, v); }
function writeF32(dv: DataView, o: number, v: number) { dv.setFloat32(o, v, true); }

export class EventBatchBuilder {
  private buf: ArrayBuffer;
  private dv: DataView;
  private p = 4;        // leave space for header
  private count = 0;

  constructor(capacity = 16 * 1024) {
    this.buf = new ArrayBuffer(capacity);
    this.dv = new DataView(this.buf);
  }

  reset() { this.p = 4; this.count = 0; }

  private ensure(extra: number) {
    if (this.p + extra <= this.buf.byteLength) return;
    const next = new ArrayBuffer((this.buf.byteLength + extra) * 2);
    new Uint8Array(next).set(new Uint8Array(this.buf, 0, this.p));
    this.buf = next;
    this.dv = new DataView(this.buf);
  }

  private beginEvent(type: EType, payloadSize: number) {
    // record header: etype(u8) + size(u16)
    this.ensure(3 + payloadSize);
    writeU8(this.dv, this.p, type);           // etype
    writeU16(this.dv, this.p + 1, payloadSize); // size
    const payloadStart = this.p + 3;
    this.p = payloadStart;
    this.count++;
    return payloadStart;
  }

  pointer(type: EType.PointerMove | EType.PointerDown | EType.PointerUp, x: number, y: number, buttons: number, mods: number, pointerId: number, kind: number) {
    const payloadSize = 16;
    const o = this.beginEvent(type, payloadSize);
    writeF32(this.dv, o + 0, x);
    writeF32(this.dv, o + 4, y);
    writeU16(this.dv, o + 8, buttons);
    writeU16(this.dv, o + 10, mods);
    writeU16(this.dv, o + 12, pointerId);
    writeU8(this.dv, o + 14, kind);
    writeU8(this.dv, o + 15, 0);
    this.p = o + payloadSize;
  }

  wheel(dx: number, dy: number, mods: number) {
    const payloadSize = 12;
    const o = this.beginEvent(EType.Wheel, payloadSize);
    writeF32(this.dv, o + 0, dx);
    writeF32(this.dv, o + 4, dy);
    writeU16(this.dv, o + 8, mods);
    writeU16(this.dv, o + 10, 0);
    this.p = o + payloadSize;
  }

  key(type: EType.KeyDown | EType.KeyUp, keyCode: number, mods: number, repeat: number) {
    const payloadSize = 8;
    const o = this.beginEvent(type, payloadSize);
    writeU16(this.dv, o + 0, keyCode);
    writeU16(this.dv, o + 2, mods);
    writeU8(this.dv, o + 4, repeat);
    writeU8(this.dv, o + 5, 0);
    writeU8(this.dv, o + 6, 0);
    writeU8(this.dv, o + 7, 0);
    this.p = o + payloadSize;
  }

  finish(): Packed {
    writeU16(this.dv, 0, this.count);
    writeU16(this.dv, 2, 0);
    return { buffer: this.buf, byteLength: this.p };
  }
}

Integration im rAF:

	•	im rAF: finish() → wasm.push_events(ptr,len) → reset()

// crates/chartcore/src/input.rs
#[derive(Copy, Clone, Debug)]
pub enum EType {
    PointerMove = 1,
    PointerDown = 2,
    PointerUp   = 3,
    Wheel       = 4,
    KeyDown     = 5,
    KeyUp       = 6,
}

#[derive(Copy, Clone, Debug, Default)]
pub struct PointerEvt {
    pub x: f32,
    pub y: f32,
    pub buttons: u16,
    pub mods: u16,
    pub pointer_id: u16,
    pub kind: u8,
    pub etype: EType,
}

#[derive(Copy, Clone, Debug, Default)]
pub struct WheelEvt {
    pub dx: f32,
    pub dy: f32,
    pub mods: u16,
}

#[derive(Copy, Clone, Debug, Default)]
pub struct KeyEvt {
    pub key_code: u16,
    pub mods: u16,
    pub repeat: u8,
    pub etype: EType,
}

pub enum InputEvt {
    Pointer(PointerEvt),
    Wheel(WheelEvt),
    Key(KeyEvt),
}

fn read_u16(buf: &[u8], o: usize) -> u16 { u16::from_le_bytes([buf[o], buf[o+1]]) }
fn read_u32(buf: &[u8], o: usize) -> u32 { u32::from_le_bytes([buf[o], buf[o+1], buf[o+2], buf[o+3]]) }
fn read_f32(buf: &[u8], o: usize) -> f32 { f32::from_le_bytes([buf[o], buf[o+1], buf[o+2], buf[o+3]]) }
fn read_u8(buf: &[u8], o: usize) -> u8 { buf[o] }

pub fn decode_events(buf: &[u8], mut f: impl FnMut(InputEvt)) {
    if buf.len() < 4 { return; }
    let count = read_u16(buf, 0) as usize;
    let mut p = 4;

    for _ in 0..count {
        if p + 3 > buf.len() { break; }
        let etype = read_u8(buf, p);
        let size = read_u16(buf, p + 1) as usize;
        let payload = p + 3;

        if payload + size > buf.len() { break; }

        match etype {
            1 | 2 | 3 => {
                if size >= 16 {
                    let x = read_f32(buf, payload + 0);
                    let y = read_f32(buf, payload + 4);
                    let buttons = read_u16(buf, payload + 8);
                    let mods = read_u16(buf, payload + 10);
                    let pointer_id = read_u16(buf, payload + 12);
                    let kind = read_u8(buf, payload + 14);
                    let et = match etype {
                        1 => EType::PointerMove,
                        2 => EType::PointerDown,
                        _ => EType::PointerUp,
                    };
                    f(InputEvt::Pointer(PointerEvt { x, y, buttons, mods, pointer_id, kind, etype: et }));
                }
            }
            4 => {
                if size >= 12 {
                    let dx = read_f32(buf, payload + 0);
                    let dy = read_f32(buf, payload + 4);
                    let mods = read_u16(buf, payload + 8);
                    f(InputEvt::Wheel(WheelEvt { dx, dy, mods }));
                }
            }
            5 | 6 => {
                if size >= 8 {
                    let key_code = read_u16(buf, payload + 0);
                    let mods = read_u16(buf, payload + 2);
                    let repeat = read_u8(buf, payload + 4);
                    let et = if etype == 5 { EType::KeyDown } else { EType::KeyUp };
                    f(InputEvt::Key(KeyEvt { key_code, mods, repeat, etype: et }));
                }
            }
            _ => {}
        }

        p = payload + size;
    }
}

WASM Export für Push:

// crates/chartcore/src/wasm_api.rs
use wasm_bindgen::prelude::*;
use crate::input::{decode_events, InputEvt};

thread_local! {
    static EVENT_RING: std::cell::RefCell<Vec<InputEvt>> = std::cell::RefCell::new(Vec::with_capacity(1024));
}

#[wasm_bindgen]
pub fn push_events(ptr: *const u8, len: usize) {
    if ptr.is_null() || len == 0 { return; }
    let buf = unsafe { std::slice::from_raw_parts(ptr, len) };

    EVENT_RING.with(|ring| {
        let mut ring = ring.borrow_mut();
        ring.clear();
        decode_events(buf, |e| ring.push(e));
    });
}

HitTest (screen-space, v1)

fn dist_point_to_segment(px:f32, py:f32, ax:f32, ay:f32, bx:f32, by:f32) -> f32 {
    let abx = bx - ax; let aby = by - ay;
    let apx = px - ax; let apy = py - ay;
    let ab2 = abx*abx + aby*aby;
    if ab2 <= 1e-6 { return ((px-ax).powi(2) + (py-ay).powi(2)).sqrt(); }
    let mut t = (apx*abx + apy*aby) / ab2;
    if t < 0.0 { t = 0.0; }
    if t > 1.0 { t = 1.0; }
    let cx = ax + t*abx; let cy = ay + t*aby;
    ((px-cx).powi(2) + (py-cy).powi(2)).sqrt()
}

fn hit_test(tr: &Trendline, x:f32, y:f32) -> Handle {
    let r = 6.0; // handle radius threshold
    let d1 = ((x-tr.x1).powi(2) + (y-tr.y1).powi(2)).sqrt();
    if d1 <= r { return Handle::P1; }
    let d2 = ((x-tr.x2).powi(2) + (y-tr.y2).powi(2)).sqrt();
    if d2 <= r { return Handle::P2; }
    let dseg = dist_point_to_segment(x,y,tr.x1,tr.y1,tr.x2,tr.y2);
    if dseg <= 4.0 { return Handle::Body; }
    Handle::None
}

Tool Event Handling (Pointer Down/Move/Up)

pub struct State {
    pub trendlines: Vec<Trendline>,
    pub tool: ToolMode,
    pub next_id: u32,
    // undo stack not shown here
}

impl State {
    fn find_tl_mut(&mut self, id: u32) -> Option<&mut Trendline> {
        self.trendlines.iter_mut().find(|t| t.id == id)
    }
    fn set_handle(&mut self, id: u32, h: Handle, p: (f32,f32)) {
        if let Some(t) = self.find_tl_mut(id) {
            match h {
                Handle::P1 => { t.x1 = p.0; t.y1 = p.1; }
                Handle::P2 => { t.x2 = p.0; t.y2 = p.1; }
                _ => {}
            }
        }
    }

    pub fn on_pointer_down(&mut self, x:f32, y:f32) {
        // if hit existing -> edit
        for tl in self.trendlines.iter().rev() { // z-order: last wins
            let h = hit_test(tl, x, y);
            if h != Handle::None && !tl.locked {
                self.tool = ToolMode::Editing { id: tl.id, dragging: h, last_x: x, last_y: y };
                return;
            }
        }

        // else create new trendline with both points at cursor
        let id = self.next_id; self.next_id += 1;
        let tl = Trendline { id, x1:x, y1:y, x2:x, y2:y, rgba:0xFFCC33FF, width:1.0, locked:false };
        // push via command in real impl
        self.trendlines.push(tl);
        self.tool = ToolMode::Creating { id, anchor_handle: Handle::P2 };
    }

    pub fn on_pointer_move(&mut self, x:f32, y:f32) {
        match self.tool {
            ToolMode::Creating { id, anchor_handle } => {
                self.set_handle(id, anchor_handle, (x,y));
            }
            ToolMode::Editing { id, dragging, last_x, last_y } => {
                let dx = x - last_x;
                let dy = y - last_y;
                if let Some(tl) = self.find_tl_mut(id) {
                    match dragging {
                        Handle::P1 => { tl.x1 = x; tl.y1 = y; }
                        Handle::P2 => { tl.x2 = x; tl.y2 = y; }
                        Handle::Body => {
                            tl.x1 += dx; tl.y1 += dy;
                            tl.x2 += dx; tl.y2 += dy;
                        }
                        _ => {}
                    }
                }
                self.tool = ToolMode::Editing { id, dragging, last_x: x, last_y: y };
            }
            _ => {}
        }
    }

    pub fn on_pointer_up(&mut self, _x:f32, _y:f32) {
        // finalize
        self.tool = ToolMode::Idle;
    }
}

Snapping v1: Ziele, Regeln, Output

Snap-Ziele (in dieser Reihenfolge)
	1.	Candle OHLC (High/Low/Close optional Open) der sichtbaren Candles
	2.	Grid / Price Steps (runde Preislevel, Ticksize)
	3.	Andere Drawings (Endpoints/Handles) – optional v1.1

Snap-Regeln
	•	Snap entscheidet in Screen Space (Pixel), damit es sich stabil anfühlt.
	•	Threshold: z.B. SNAP_PX = 6.0
	•	Modifier:
	•	Alt = Snap aus
	•	Shift = Axis-Lock (nur X oder nur Y; abhängig von Drag-Richtung oder Handle-Typ)

Snap-Output (für UI + Rendering)

Rust liefert pro Frame:
	•	snap_active: bool
	•	snap_x: f32, snap_y: f32 (screen coords)
	•	snap_kind: u8 (OHLC=1, GRID=2, DRAWING=3)
	•	optional: snap_label_text_id (“H”, “L”, “C”, “Grid”)

Das kommt idealerweise in FrameMeta oder als separater get_interaction_state().

⸻

Algorithmus: schnell + sauber

1) Candidate-Liste aufbauen (nur wenn nötig)

Du willst NICHT bei jedem Mousemove alle Candles scannen.

Trick: Für OHLC brauchst du fast immer nur die Candle unter dem Cursor + Nachbarn.
	•	ScreenX → DataIndex über Viewport Mapping
	•	Prüfe Candle i, i-1, i+1 (oder +-2)
	•	Kandidatenpunkte: (x_candle, y_high), (x_candle, y_low), (x_candle, y_close) (+ open optional)

Dann Grid:
	•	price = screen_to_price(y)
	•	grid_price = round_to_step(price, step) (Ticksize oder “nice step”)
	•	y_grid = price_to_screen(grid_price)
	•	Candidate (x, y_grid) (nur y snap) oder (x_grid, y) (nur x snap, wenn time grid)

2) Scoring (Screen-Distanz)

Für jeden Kandidaten:
	•	dx = cand.x - x
	•	dy = cand.y - y
	•	dist2 = dx*dx + dy*dy
	•	nehme min dist2, wenn sqrt(dist2) <= SNAP_PX

Axis lock:
	•	Wenn Shift: setze dx=0 oder dy=0 (je nach Lock), oder ignoriere Kandidaten, die die gesperrte Achse ändern.

3) Anwenden
	•	Wenn Snap aktiv: verwende (snap_x, snap_y) als “effective pointer”
	•	Wenn nicht: verwende (x, y)

Rust: Snap-Funktion (v1, OHLC + Grid)

const SNAP_PX: f32 = 6.0;

fn snap_point(
    state: &State,
    x: f32,
    y: f32,
    mods: u16,
    axis_lock: AxisLock,
) -> SnapResult {
    // Alt disables snap
    if (mods & MOD_ALT) != 0 {
        return SnapResult { active:false, x, y, kind:SnapKind::None, label:0 };
    }

    let mut best = SnapResult { active:false, x, y, kind:SnapKind::None, label:0 };
    let mut best_d2 = SNAP_PX * SNAP_PX;

    // ----- OHLC snap: only nearby candle(s) -----
    if let Some(i) = state.viewport.screen_x_to_index(x) {
        for j in [i.saturating_sub(1), i, i+1] {
            if let Some(c) = state.candles.get(j) {
                let cx = state.viewport.index_to_screen_x(j);

                // Map prices to screen y
                let yh = state.viewport.price_to_screen_y(c.high);
                let yl = state.viewport.price_to_screen_y(c.low);
                let yc = state.viewport.price_to_screen_y(c.close);

                // candidates
                for (cy, label) in [(yh, state.text_id_h), (yl, state.text_id_l), (yc, state.text_id_c)] {
                    let (px, py) = apply_axis_lock(x, y, cx, cy, axis_lock);
                    let dx = px - x;
                    let dy = py - y;
                    let d2 = dx*dx + dy*dy;
                    if d2 <= best_d2 {
                        best_d2 = d2;
                        best = SnapResult { active:true, x:px, y:py, kind:SnapKind::Ohlc, label };
                    }
                }
            }
        }
    }

    // ----- Grid snap (Y only) -----
    let price = state.viewport.screen_y_to_price(y);
    let step = state.viewport.grid_step_price(); // ticksize or "nice"
    let gp = (price / step).round() * step;
    let gy = state.viewport.price_to_screen_y(gp);

    let (px, py) = apply_axis_lock(x, y, x, gy, axis_lock);
    let dx = px - x;
    let dy = py - y;
    let d2 = dx*dx + dy*dy;
    if d2 <= best_d2 {
        best = SnapResult { active:true, x:px, y:py, kind:SnapKind::Grid, label: state.text_id_grid };
    }

    best
}

#[derive(Copy, Clone)]
pub enum AxisLock { None, X, Y }

fn apply_axis_lock(x:f32, y:f32, candx:f32, candy:f32, lock: AxisLock) -> (f32,f32) {
    match lock {
        AxisLock::None => (candx, candy),
        AxisLock::X => (candx, y),   // lock Y, allow X changes
        AxisLock::Y => (x, candy),   // lock X, allow Y changes
    }
}

Hinweis: AxisLock kannst du aus Shift + Drag-Direction ableiten (z.B. sobald |dx| > |dy| → lock Y, sonst lock X), oder je Tool/Handle festlegen.

Beim Drag eines Handles:

pub fn on_pointer_move(&mut self, x:f32, y:f32, mods:u16) {

Sticky Snap v1: Konzept

Du führst einen kleinen “SnapController”, der merkt:
	•	welches Target gerade “gefangen” ist (candle index + ohlc-type / grid / drawing)
	•	ob er noch “drin” ist (exit threshold)
	•	wann er wieder “frei” wird

Regeln:
	•	Enter: neues Target darf “einrasten”, wenn Distanz ≤ ENTER_PX (z.B. 6)
	•	Hold: wenn ein Target aktiv ist, bleibt es aktiv solange Distanz ≤ EXIT_PX (z.B. 10)
	•	Switch: nur wenn ein anderes Target deutlich besser ist (optional “switch margin”)

SnapController (Sticky Logic)

pub struct SnapController {
    pub enter_px: f32,
    pub exit_px: f32,
    pub switch_margin_px: f32, // optional, z.B. 2.0
    current: SnapResult,
}

impl SnapController {
    pub fn new() -> Self {
        Self {
            enter_px: 6.0,
            exit_px: 10.0,
            switch_margin_px: 2.0,
            current: SnapResult { active: false, id: SnapTargetId::None, x: 0.0, y: 0.0, label: 0, kind: 0 },
        }
    }

    pub fn clear(&mut self) {
        self.current.active = false;
        self.current.id = SnapTargetId::None;
        self.current.kind = 0;
        self.current.label = 0;
    }

    pub fn current(&self) -> SnapResult { self.current }

    pub fn update(&mut self, pointer_x: f32, pointer_y: f32, candidates: &[SnapCandidate]) -> SnapResult {
        let enter_d2 = self.enter_px * self.enter_px;
        let exit_d2  = self.exit_px  * self.exit_px;
        let switch_margin_d2 = self.switch_margin_px * self.switch_margin_px;

        // helper: distance pointer -> (x,y)
        let dist2 = |x: f32, y: f32| {
            let dx = x - pointer_x;
            let dy = y - pointer_y;
            dx*dx + dy*dy
        };

        // 1) Wenn wir ein aktuelles Target haben: prüfen, ob wir es halten
        if self.current.active {
            let cur_d2 = dist2(self.current.x, self.current.y);

            // Wenn noch innerhalb EXIT: halten, aber optional besseres Target zulassen
            if cur_d2 <= exit_d2 {
                // optionaler Switch: nur wenn ein anderes Target "signifikant" besser ist
                if let Some(best) = candidates.iter().min_by(|a,b| a.d2.total_cmp(&b.d2)) {
                    // switch nur wenn best wirklich näher ist als current minus margin
                    if best.d2 + switch_margin_d2 < cur_d2 && best.d2 <= enter_d2 {
                        self.current = SnapResult {
                            active: true, id: best.id, x: best.x, y: best.y, label: best.label, kind: best.kind
                        };
                    }
                }
                return self.current;
            }

            // Außerhalb EXIT → release und weiter unten neu einrasten lassen
            self.clear();
        }

        // 2) Kein aktuelles Target: neues Target einrasten lassen, wenn <= ENTER
        if let Some(best) = candidates.iter().min_by(|a,b| a.d2.total_cmp(&b.d2)) {
            if best.d2 <= enter_d2 {
                self.current = SnapResult {
                    active: true, id: best.id, x: best.x, y: best.y, label: best.label, kind: best.kind
                };
                return self.current;
            }
        }

        self.current
    }
}

Kandidaten erzeugen (stabil identifizieren)

Candle OHLC Kandidaten (nur i-1,i,i+1)

Wichtig: Der TargetId ist stabil: {index, point}.

fn candle_ohlc_candidates(
    state: &State,
    x: f32,
    y: f32,
    out: &mut Vec<SnapCandidate>,
) {
    let Some(i) = state.viewport.screen_x_to_index(x) else { return; };

    for j in [i.saturating_sub(1), i, i + 1] {
        if let Some(c) = state.candles.get(j as usize) {
            let cx = state.viewport.index_to_screen_x(j);

            let points = [
                (OhlcPoint::High,  state.viewport.price_to_screen_y(c.high),  state.text_id_h),
                (OhlcPoint::Low,   state.viewport.price_to_screen_y(c.low),   state.text_id_l),
                (OhlcPoint::Close, state.viewport.price_to_screen_y(c.close), state.text_id_c),
                // optional:
                // (OhlcPoint::Open,  state.viewport.price_to_screen_y(c.open),  state.text_id_o),
            ];

            for (pt, cy, label) in points {
                let dx = cx - x;
                let dy = cy - y;
                let d2 = dx*dx + dy*dy;

                out.push(SnapCandidate {
                    id: SnapTargetId::CandleOhlc { index: j, point: pt },
                    x: cx,
                    y: cy,
                    label,
                    kind: 1,
                    d2,
                });
            }
        }
    }
}

Grid Kandidat (Y-only, mit “level id”)

Damit Grid-Snap sticky ist, braucht es eine stabile ID. Einfach: level = round(price/step).

fn grid_y_candidate(state: &State, x: f32, y: f32, out: &mut Vec<SnapCandidate>) {
    let price = state.viewport.screen_y_to_price(y);
    let step  = state.viewport.grid_step_price();
    if step <= 0.0 { return; }

    let level = (price / step).round() as i32;
    let gp = (level as f32) * step;
    let gy = state.viewport.price_to_screen_y(gp);

    let dx = 0.0;
    let dy = gy - y;
    let d2 = dx*dx + dy*dy;

    out.push(SnapCandidate {
        id: SnapTargetId::GridY { level },
        x,
        y: gy,
        label: state.text_id_grid,
        kind: 2,
        d2,
    });
}

Snap Integration in deinem Pointer-Move

pub fn on_pointer_move(&mut self, x: f32, y: f32, mods: u16) {
    // Alt disables snap
    if (mods & MOD_ALT) != 0 {
        self.snap_controller.clear();
        self.interaction.snap = SnapResult { active:false, id:SnapTargetId::None, x, y, label:0, kind:0 };
        self.apply_pointer(x, y, mods);
        return;
    }

    // 1) Kandidaten bauen
    self.tmp_candidates.clear();
    candle_ohlc_candidates(self, x, y, &mut self.tmp_candidates);
    grid_y_candidate(self, x, y, &mut self.tmp_candidates);

    // 2) Sticky snap update
    let snap = self.snap_controller.update(x, y, &self.tmp_candidates);
    self.interaction.snap = snap;

    // 3) Effective pointer coords
    let (ex, ey) = if snap.active { (snap.x, snap.y) } else { (x, y) };

    self.apply_pointer(ex, ey, mods);
}

apply_pointer(ex, ey, mods) ist deine bestehende Tool-Logik (Creating/Editing).

Snapping-Policy nach Drag-Mode

A) Handle-Drag (P1/P2)
	•	Snap in 2D (X+Y) an Candle-OHLC (starkes Feature)
	•	Grid-Snap eher Y-only (Preislevel), optional auch X-only an Time-grid
	•	Shift: Axis-Lock (hart)

B) Body-Drag (Body)
	•	Snap primär Y-only (Preislevel / OHLC), X bleibt frei (sonst “klebt” die Linie unangenehm)
	•	Candle-OHLC Snap:
	•	bei Body meist nicht auf Candle-X springen
	•	eher nur auf Preis (y) bzw. “nearest price level”
	•	Optional: wenn Nutzer nahe an Candle-OHLC ist, snappt nur Y auf OHLC-Wert, nicht X.

Merksatz: Handles = “an Daten ankern”, Body = “Objekt verschieben”.

⸻

Data-space vs Screen-space: Was ist was?

Du hast intern zwei Koordinatensysteme:
	•	Screen space: px; gut für Distanz (Snap threshold)
	•	Data space: (time/index, price); gut für “was bedeutet das Objekt”

Wichtig: Du kannst in screen space entscheiden ob du snappen willst, aber die resultierende Bewegung sollte in data space stabil bleiben.

⸻

Konkrete Umsetzung: “Snap Axis Policy”

Wir definieren pro Drag-Mode:

#[derive(Copy, Clone)]
pub enum SnapAxisPolicy {
    Free2D,     // handles: X und Y dürfen vom Snap überschrieben werden
    YOnly,      // body: nur Y darf snappen, X bleibt pointer-getrieben
    XOnly,      // selten, z.B. vertical line tool
}

Und pro Drag-Mode wählst du:
	•	Handle P1/P2: Free2D
	•	Body: YOnly

⸻

Kandidaten: Candle OHLC für Y-only vs 2D

1) OHLC Candidate für Handle (2D)
	•	candidate.x = candle_x
	•	candidate.y = ohlc_y
	•	d2 = (cx-x)^2 + (cy-y)^2

2) OHLC Candidate für Body (Y-only)
	•	candidate.x = pointer_x (NICHT candle_x)
	•	candidate.y = ohlc_y
	•	d2 = (ohlc_y - y)^2

Das ist die gesamte Magie. Du willst die Preisinformation nutzen, aber nicht die Zeitachse “reißen”.

Beispiel:

fn push_ohlc_candidate(
    out: &mut Vec<SnapCandidate>,
    id: SnapTargetId,
    policy: SnapAxisPolicy,
    pointer_x: f32,
    pointer_y: f32,
    candle_x: f32,
    ohlc_y: f32,
    label: u32,
    kind: u8
) {
    let (sx, sy, d2) = match policy {
        SnapAxisPolicy::Free2D => {
            let dx = candle_x - pointer_x;
            let dy = ohlc_y - pointer_y;
            (candle_x, ohlc_y, dx*dx + dy*dy)
        }
        SnapAxisPolicy::YOnly => {
            let dy = ohlc_y - pointer_y;
            (pointer_x, ohlc_y, dy*dy)
        }
        SnapAxisPolicy::XOnly => {
            let dx = candle_x - pointer_x;
            (candle_x, pointer_y, dx*dx)
        }
    };

    out.push(SnapCandidate { id, x: sx, y: sy, label, kind, d2 });
}

Body-Drag: Bewegung in Data-space statt Screen-space

Wenn du Body ziehst, ist es extrem wichtig, dass die Linie nicht “driftet” durch Pixel-Rundung.

Pattern:
	1.	Bei PointerDown auf Body: speichere den Anker in Data-Space
	•	grab_time = screen_to_time(x)
	•	grab_price = screen_to_price(y)
	•	und speichere die Trendline-Punkte in Data-Space (oder Screen-Space + inverse transform stabil)
	2.	Bei Move:
	•	compute delta in Data:
	•	dt = current_time - grab_time
	•	dp = current_price - grab_price
	•	apply to both endpoints in Data space
	3.	Snap: nur dp (Y) wird angepasst (Body policy Y-only)

Konkreter Ablauf
	•	erst raw_dt, raw_dp aus pointer
	•	dann optional snapped_dp aus Snap (Y-only)
	•	dann endpoints = base_endpoints + (raw_dt, snapped_dp)

Damit bleibt X “smooth” und Y kann magnetisch werden.

1) Drei Koordinatenräume (klar trennen)

A) Data Space (wichtigster)

	•	Preis: p (f64/f32)

C) View Space (optional)

2) Entscheidung: Zeit als bar_index vs timestamp

Du hast Realtime/Backfill und verschiedene Timeframes. Das ist wichtig:

Option 1: bar_index (empfohlen für MVP)

Vorteile
	•	super stabil in einem Chart (Index→X ist eindeutig)
	•	HitTest und snapping zu Candles trivial
	•	klein, schnell, deterministisch

Nachteile
	•	wenn Daten “neu eingefügt” werden (Backfill, missing bars), kann Index verschieben
	•	Cross-symbol / timeframe wird schwieriger

Option 2: timestamp_ms (langfristig besser)

Vorteile
	•	stabil über Backfills/Resamples
	•	Drawing Sync über Charts / Multi-Timeframe leichter

Nachteile
	•	braucht Mapping timestamp→x (und Interpolation)
	•	bei Märkten ohne exakte bars (irregulär) musst du sauber mappen

Pragmatischer Hybrid (mein Vorschlag)

Speichere beides, aber mit Priorität:

pub enum TimeAnchor {
    Index(u32),
    Timestamp(i64),
}

Und persistierst: Timestamp + optional IndexHint.
Runtime nutzt IndexHint, solange gültig; wenn Daten verschoben: re-resolve via timestamp.

Das ist genau die Art “stabil bei Backfill”, ohne MVP zu verkomplizieren.
